Overview

This repository contains demos showcasing Amazon Nova's capabilities in image inpainting and image variation. These demos demonstrate how Nova can intelligently modify and generate images by filling in missing parts (inpainting) or creating new variations of an existing image while maintaining visual coherence.

Features

Image Inpainting: Fill in missing or removed sections of an image with contextually relevant details.

Image Variation: Generate alternative versions of an image while preserving its essential features.
